# Call-Center-Performance-Overview
This project demonstrates the creation of an interactive Power BI dashboard designed to visualize key performance indicators (KPIs) for call center operations. The dashboard is focused on providing actionable insights into call center performance, to improve decision-making and overall efficiency. By leveraging Power BI's powerful data visualization capabilities, this dashboard presents critical metrics such as customer satisfaction, call handling times, and agent performance.

## Features

- **Customer Satisfaction**: Visualizes the customer satisfaction trends and insights.
- **Calls Answered/Abandoned**: Tracks the total number of calls answered and abandoned.
- **Calls by Time**: Displays how calls are distributed across different times of the day.
- **Average Speed of Answer**: Calculates and displays the average time taken to answer calls.
- **Agent's Performance Quadrant**: A scatter plot visual comparing agent performance based on average handle time vs calls answered.
